# Fighter

A fighter plane battle game built with PyGame.

## Installation

```bash
uv add fighter
```

## Usage

### Command Line

```bash
# Run the game
fighter
```

### Game Controls

- **Arrow Keys** or **A/D**: Move left/right
- **Space**: Shoot
- **P**: Pause/Resume
- **R**: Restart (after game over)
- **ESC**: Exit

## Features

- Three difficulty levels (Easy, Medium, Hard)
- Multiple enemy types (Fast, Strong, Boss)
- Power-ups (Attack Boost, Health, Rapid Fire, Shield, Multi Shot)
- Score system and countdown timer
- Three-player formation (three planes that move together)

## Development

The game uses PyGame for graphics and input handling. The code is organized with classes for game entities (Player, Enemy, Bullet, PowerUp) and a main Game class that manages the game loop.
